Some Vocab

Understanding the basics

Parentheses ()
Parentheses are used predominantly for housing statements within an event, function, loop, etc. They are also used for calling functions (when empty) or housing parameters or arguments when writing functions. When writing an expression with elements that include a space bar character, you can also use parentheses to wrap this element so it is recognised as a single unit.

Brackets []
Brackets are used to wrap arrays, or objects from arrays when extracted for use in other statements.

Braces {}
Braces are used for wrapping multiple statements that make up the body of a function, especially including nested functions or events. They can also be used to to wrap objects in an array if they have numerous properties.

Single quotes '' and Double quotes ""
I'm not entirely sure what the difference is between the uses of single and double quotes; they seem mostly interchangeable. But in my coding so far, single quotes are used to wrap classes and event listeners, whereas double quotes are used for strings.
