Omg Javascript...

Javascript compared to HTML & CSS

Wow! I finally hit javascript! Or, really, it hit me. It's an interesting one, and as such an intuitive thinker I found it waaay harder to get my head around. I haven't done maths like that since NCEA level 2. It feels much more like I'm speaking directly to a computer than html or css, which I guess makes sense given the increased level of involvement between the computer and the commands I'm writing. I guess the main difference I see is that html and css languages are mostly descriptive in nature, whereas js is strictly logic and mathematics.
